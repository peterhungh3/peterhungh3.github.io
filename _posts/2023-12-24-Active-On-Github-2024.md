I was never active on Github. That may sound strange to many, especially given that I've been in software for almost 1.5 decade. 

Before, I was so busy writing close-sourced software products that I didn't have the time and mind to think about being active on Github. Many of the things I built were/are free to use as a product (such as this old <a href="https://vietnameseaccent.com/?r=ph-gh" target="_blank">AI tool</a>), but they weren't open-sourced.

In 2024, as <a href="/about/#back-to-school" target="_blank">I'm back to school</a> and building a new career path, I set a goal to become more active on Github. Given my workload, I want to focus on simple, yet helpful, actions that I can maintain to build up my muscle strength of being active on Github. Some of these actions include: 
- Share useful AI tools and papers. And write comment on them. 
- Star the repos I like.
- Continue to update and add new content to this Github blog. (And although I'm familiar with Wordpress, which I used for my main blog, I quite enjoyed writing posts like this on Github pages using markdown and Vim.)


Hope to see you around and keep me accountable at this Github blog and <a href="https://github.com/peterhungh3" target="_blank">my Github repo</a>.
